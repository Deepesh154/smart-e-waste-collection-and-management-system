# Smart E-Waste Collection and Management System

Final Year Project

## Description
A web-based system to manage e-waste collection, pickup, and disposal using users, collectors, and admin workflows.

## Tech Stack
- Frontend: HTML, Css, Js , React.js, Tailwind.js
- Backend: Node.js / Express
- Database: MySQL
